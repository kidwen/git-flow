clone your repository. 
					
						git clone git@github.com:kidwen/git-flow.git
						// or
						git clone https://github.com/kidwen/git-flow.git
Check the branches and switch another branch. 
					
						git branch -a
Then you can switch to other branch if the branch exited. 
						
							git checkout dev
							// or
							git switch dev
Or create a new branch base on this branch by use params `-b`. 
						
							git checkout -b dev
							// or
							git switch -c dev
How to push your code to remote respository? 
					
						# check witch file has been changed
						git status
						# add all files
						git add .
						# set this commit message
						git commit -m "commit message"
						# push code to remote repository
						git push -u origin dev
Rebase code.
If you current branch is dev, and you want rebase to master. 
					
						# fetch the latest changes form remote git repository.
						git fetch
						# integrate changes from one branch into another
						git rebase origin/master
						# if you have a conflicts, resolve it and use next command
						git rebase --continue
						# or you want to cancel the rebasing
						git rebase --abort
